## 1. 协议双方

- **许可方**：商品发布者（NiceShoply 官方或第三方开发者，以下称「开发者」）

本协议在你完成购买并下载/安装商品时生效。

## 2. 授权类型

### 2.1 标准授权（Standard License）

| 项目 | 说明 |
|------|------|
| 购买方式 | 一次性付费 |
| 使用权 | **永久**——已安装版本可持续使用 |
| 更新权 | 含购买后 **365 天**（或商品标注天数）技术支持与版本更新 |
| 过期后 | 已装版本可继续运行；**不可**下载新版本 |
| 续费 | 可续费技术支持期以恢复更新下载权限 |

### 2.2 订阅授权（Subscription License）

| 项目 | 说明 |
|------|------|
| 购买方式 | 按月/年订阅 |
| 使用权 | 订阅有效期内可用 |
| 过期后 | **功能不可用**（适用于 SaaS 类、API 类插件） |
| 验证 | 由开发者服务端 + 市场平台联合验证 |

### 2.3 域名授权档位

商品可能限制绑定域名数量，常见档位：

- **单域名**：1 个生产域名
- **多域名**：最多 3 个生产域名
- **无限**：不限制生产域名数量（不含转售授权）

具体档位以商品详情页为准。
