# NiceShoply 授权协议体系

> 本文档说明 NiceShoply 的完整授权架构。**核心卖点：你拥有完整源码，数据与店铺真正归你所有**——这与 Shopify 等 SaaS 租用模式有本质区别。

## 与 Shopify 的核心差异

| 维度 | Shopify | NiceShoply |
|------|---------|------------|
| **源码** | 不开放，黑盒租用 | **完整源码开放（MIT），可下载、修改、自托管** |
| **数据归属** | 平台托管，迁移受限 | **数据库与文件在你自己的服务器** |
| **定制自由** | 受主题/Liquid 与 App 限制 | **可改任意 PHP/前端代码，Hook 扩展** |
| **平台抽成** | 交易抽成 + 月费 | **零 GMV 抽成**（自托管免费） |
| **停运风险** | 平台规则/封号即失联 | **源码在握，可独立运行** |
| **付费点** | 月费、抽成、App 订阅 | **可选**：托管云、增值插件、商业授权 |

> **一句话**：Shopify 是「租商城」；NiceShoply 是「把商城买回家，源代码一并给你」。

```
│  → 文档：[开源核心许可说明](./开源核心许可说明.md)            │
└─────────────────────────────────────────────────────────────┘
                              │
          ┌───────────────────┼───────────────────┐
          ▼                   ▼                   ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│ 第二层：应用市场  │ │ 第三层：商业授权  │ │ 服务层：托管云   │
│ 插件/主题 EULA   │ │ 去版权/白标/企业  │ │ SaaS 服务协议    │

```

**原则**：开源协议不承载收费逻辑；收费通过 **EULA（商品许可）** 和 **商业合同（品牌/分发权）** 实现。

## 各层适用范围

### 第一层：开源核心（MIT）

**永久免费**，包含但不限于：

- 完整商城前台、后台控制台、REST API
- 商品/订单/结账/客户/CMS/评价/税务等核心电商功能
- 多语言多币种、安装向导、插件/主题机制
- 基础支付（BankTransfer 等）、基础物流（FixedShipping）
- DevTools 脚手架（`make:plugin` / `make:theme`）

**你的权利**（MIT 赋予）：

- ✅ 商用、修改、二次开发、私有化部署
- ✅ Fork、再分发（须保留版权声明）
- ✅ 无需向官方付费即可正常开店运营
- ❌ 不得使用 NiceShoply 商标暗示官方背书（见 [TRADEMARK.md](../../niceshoply/TRADEMARK.md)）

许可文件：[`niceshoply/LICENSE`](../../niceshoply/LICENSE)

---

### 第二层：应用市场商品（EULA）

通过 [marketplace.niceshoply.com](https://marketplace.niceshoply.com) 购买的**付费插件、主题**适用独立 EULA，与核心 MIT 许可分离。

| 类型 | 购买方式 | 过期后 |
|------|----------|--------|
| **标准授权** | 一次性买断 + 含 N 天更新/support | 已装版本可继续用，不可下新版 |
| **订阅授权** | 月/年订阅 | 到期功能不可用 |

详见：[插件与主题 EULA](./插件与主题EULA.md)

---

### 第三层：商业授权（合同）

面向「拿去商用分发」场景，**不影响普通自托管用户**：

| 档位 | 适用对象 | 核心权益 |
|------|----------|----------|
| 去版权授权 | 单站商用 | 移除 Powered by、自定义品牌 |
| 服务商/白标授权 | 建站公司、代理 | 多客户交付、白标、伙伴标识 |
| 企业私有化授权 | 中大型企业 | 定制条款、SLA、专属支持 |

详见：[商业授权协议](./商业授权协议.md)

---

### 服务层：NiceShoply Cloud

托管云是 **SaaS 服务**，不是源码许可：

- 卖的是省心：自动备份、升级、监控、HTTPS、邮件送达
- 不锁核心功能；用户可随时迁出自托管
- 服务条款见官网 Cloud 页面（上线时发布）

---

## 收费边界（写死，避免反复横跳）

1. **不阉割核心**：不付费也能完整开店。
2. **对「省心」收费**：Cloud 托管订阅。
3. **对「深度能力」收费**：官方高级插件/主题。
4. **对「商用分发权」收费**：去版权/白标授权。
5. **开源协议不加使用限制**：不用协议条款强留应用市场或品牌。
