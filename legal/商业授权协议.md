# 商业授权协议

> **适用范围**：购买 NiceShoply **去版权、白标、企业私有化**等商业授权的客户。  
> **不适用于**：普通自托管用户（MIT 已允许免费商用开店，无需本协议）。

---

## 1. 为什么需要商业授权

NiceShoply 主程序以 MIT 开源，**完整源码免费可商用**。商业授权面向的是「**商用分发与品牌层面**」的增值需求，而非「使用软件」本身：

| 需求 | 是否需要商业授权 |
|------|------------------|
| 自己开店卖货 | ❌ 不需要 |
| 自托管、改代码、接插件 | ❌ 不需要 |
| 移除「Powered by NiceShoply」 | ✅ 需要 **去版权授权** |
| 以自有品牌向多个客户交付建站 | ✅ 需要 **白标授权** |
| 企业私有化 + SLA + 定制条款 | ✅ 需要 **企业授权** |

> 与 Shopify 不同：Shopify 按月租用且无源码；NiceShoply 源码免费，商业授权只对**品牌与分发权**收费。

## 2. 授权档位

### 2.1 去版权授权（Branding Removal License）

**适用**：单一生产站点，希望移除官方品牌露出。

| 项目 | 内容 |
|------|------|
| 参考价 | $99–$199 / 站 / 年（或一次性买断，以官网为准） |
| 授权域名 | 1 个生产域名 |
| 权益 | 移除前台/后台「Powered by NiceShoply」及官方品牌链接 |
| 权益 | 可替换为自有品牌名称与链接 |
| 不含 | 源码再分发权、多客户白标 |

**默认行为（未购买时）**：自托管实例可在页脚展示「Powered by NiceShoply」链接——这是 MIT 允许的品牌署名，也帮助社区传播。

---

### 2.2 服务商 / 白标授权（Agency / White-Label License）

**适用**：建站公司、自由开发者、代理商，以 NiceShoply 为底座向**多个客户**交付。

| 项目 | 内容 |
|------|------|
| 参考价 | $499–$999 / 年 |
| 授权范围 | 许可方（服务商）名下的多个客户项目 |
| 权益 | 去版权 + 白标交付（对客户展示服务商品牌） |
| 权益 | 合作伙伴标识与官网展示（可选） |
| 权益 | 优先技术支持通道 |
| 权益 | Cloud 多店铺 / 子账户折扣（如有） |
| 限制 | 不得将 NiceShoply 核心源码作为独立产品转售 |
| 限制 | 不得申请「NiceShoply 官方」名义 |

---

### 2.3 企业私有化授权（Enterprise License）

**适用**：中大型企业，需要私有化部署、定制法律条款与 SLA。

| 项目 | 内容 |
|------|------|
| 参考价 | $1,999+ / 年，面议 |
| 权益 | 包含白标授权全部权益 |
| 权益 | 定制化许可条款（域名数、子公司、内网部署等） |
| 权益 | SLA 响应时间约定 |
| 权益 | 专属技术支持联系人 |
| 权益 | 安全通告优先通知 |

## 3. 授权激活与核验

1. 在 [marketplace.niceshoply.com](https://marketplace.niceshoply.com) 或官网购买对应授权商品
2. 绑定生产域名，获取 **License Key** / **Domain Token**
3. 在后台「授权管理」中激活
4. 系统核验通过后，开放对应权益（如隐藏 Powered by）

**授权与源码的关系**：

- 授权控制的是**品牌展示与商业分发权益**
- **不**限制你访问、修改、备份 MIT 开源源码
- 授权到期后：品牌限制恢复默认（须重新展示 Powered by），**已部署源码与数据不受影响**

---

## 4. 许可限制（所有档位通用）

被许可方**不得**：

1. 将授权转让、转售给第三方（企业授权中另有书面约定的除外）
2. 使用 NiceShoply 商标暗示官方背书或官方认证
3. 将授权密钥共享至未授权域名
4. 声称拥有 NiceShoply 核心著作权

## 6. 期限与续费

- 年费授权：到期前 30 天提醒续费
- 续费：从当前到期日顺延
- 过期：品牌权益收回；**不**远程删除你的代码或数据
- 买断型（如有）：以购买时约定为准

---

## 7. 退款与取消

- 授权激活并核验成功后，原则上不退款
- 未激活且 7 日内，可申请审核退款
- 服务商授权可约定年度最低承诺，提前取消按剩余比例处理（以订单条款为准）

## 附录：Shopify 卖家迁移说明

若你从 Shopify 迁移至 NiceShoply 并购买商业授权：

1. **源码**：首次获得完整 PHP/Vue 源码控制权
2. **数据**：订单/客户/商品数据导入你自有数据库
3. **品牌**：购买去版权/白标授权后，对客户完全以你的品牌交付
4. **成本**：零 GMV 抽成 + 可选 Cloud/插件，总拥有成本（TCO）显著低于 SaaS 租用

这正是 NiceShoply 的核心价值主张：**不是租商城，是把商城真正买回家。**
